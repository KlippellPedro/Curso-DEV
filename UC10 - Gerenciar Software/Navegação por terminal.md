### Navegação por terminal



**dir** = ver oque tem dentro da pasta

**cd** + nome da pasta = entra na pasta

**cd..** = sair de dentro da pasta

**cls** = limpa os comandos

**git clone** + link do repositório copiado no git hub = coloca o arquivo no PC direto do GitHub

**git add .**  = adiciona todas as alterações feitas no git

**git commit --m** = usado para anotar as alterações feitas no código

**git push** = importo os arquivos para o repositório no git

**git pull** = puxa os arquivos do repositório do git pro PC
